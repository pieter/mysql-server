Test suite FAQ

  1. The test suite has been running for over a day. What's wrong?

    The test suite an take anywhere from some number of hours to several days to run, depending on your hardware configuration. As long as the run is making forward progress and new lines are being written to the ALL.OUT file, everything is probably fine.

  2. The test suite hangs.

    The test suite requires Tcl 8.1 or greater, preferably at least Tcl 8.3. If you are using an earlier version of Tcl, the test suite may simply hang at some point.
